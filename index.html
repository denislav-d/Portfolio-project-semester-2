User Centered Design

In the User Centered Design module, I learn how to research, test, and design interactive media products that meet the needs of my clients and users. The primary learning outcome is to create prototypes of interactive media products using iterations, user research, and examples from the professional field. Prototypes are designed to be user-friendly with techniques ranging from low-fidelity paper sketches to high-fidelity by using tools like Figma.
